# ESLint config file for NestJS projects

## Installation

With `yarn`:

```bash
yarn add -D @sotream/eslint-config-nestjs
```

With `npm`:

```bash
npm i -D @sotream/eslint-config-nestjs
```

## Usage

Create `.eslintrc.js` file:

```js
module.exports = {
  "extends": [
    "@sotream/eslint-config-nestjs"
  ],
  "rules": {
      // overrides ...
  }
}
```
