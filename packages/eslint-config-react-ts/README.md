# ESLint config file for React TypeScript projects

## Installation

With `yarn`:

```bash
yarn add -D @sotream/eslint-config-react-ts
```

With `npm`:

```bash
npm i -D @sotream/eslint-config-react-ts
```

## Usage

Create `.eslintrc.js` file:

```js
module.exports = {
  "extends": [
    "@sotream/eslint-config-react-ts"
  ],
  "rules": {
    // overrides ...
  }
}
```
